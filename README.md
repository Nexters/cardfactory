# cardfactory
Cardfactory project

# 세팅
- 노드 설치 (https://nodejs.org/en/download/)

- npm 글로벌 모듈 설치

```
    $ npm install -g express
    $ npm install -g bower
```

- npm, bower 모듈 설치

```
    $ cd cardfactory
    $ npm install
    $ bower install
```

- 시작하려면 터미널 2개 띄워서

```
    $ node ./bin/www
    $ gulp watch
```

- 하나로 실행

```
    $ npm start
```
